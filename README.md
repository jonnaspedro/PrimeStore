# Modelo ER de E-Commerce

Este projeto apresenta o diagrama Entidade-Relacionamento de um sistema de e-commerce. A estrutura foi organizada para representar um site capaz de gerenciar clientes, produtos, categorias e pedidos de forma integrada.

## Conteúdo
- Diagrama ER do sistema
- Entidades principais: Clientes, Produtos, Categorias, Pedidos e Itens do Pedido
- Relações entre as entidades e estrutura básica do banco de dados

## Objetivo
Fornecer uma visão clara e organizada da base de dados de um e-commerce, servindo como referência para estudos, documentação ou implementação em projetos reais.

## Como usar
Consulte o diagrama disponível no repositório para entender a estrutura e os relacionamentos do sistema.
